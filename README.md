# Ginka

Bilibili Live Command-line Tool. ( MacOS only )

## Usage

You should set your bilibili cookie as `COOKIE` variable at first.

### Live Mode ( Start Live Stream )
```Bash
ginka -l <ROOM_ID>
```

Then you will get your rtmp address and code.
